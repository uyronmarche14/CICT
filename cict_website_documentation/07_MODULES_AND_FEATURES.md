# CICT Website Modules and Features

## 1. Authentication Module

### Purpose
Allow secure access to admin accounts.

### Features

- Admin login
- Admin logout
- Session protection
- Profile retrieval

---

## 2. Dashboard Module

### Purpose
Show a quick system overview to admins.

### Features

- Total users
- Total news
- Total announcements
- Total roles

---

## 3. User Management Module

### Purpose
Manage admin users.

### Features

- Create user
- View users
- Search users
- Paginate users
- Delete user
- Role display
- Active status display

---

## 4. Roles and Permissions Module

### Purpose
Control access to protected features.

### Features

- Full Admin role
- Semi Admin role
- Support role
- Permission checks
- Role listing
- Custom role model support

---

## 5. News Module

### Purpose
Publish official article-style content.

### Features

- Create news
- Edit news
- Delete news
- Upload featured image
- Add tags
- Set status
- Public news listing
- Public news detail page

### Content Types for News

- department updates
- achievements
- event coverage
- competition highlights
- faculty recognitions
- student features

---

## 6. Announcement Module

### Purpose
Publish short official notices and reminders.

### Features

- Create announcement
- Edit announcement
- Delete announcement
- Set priority
- Set target audience
- Set expiration
- Upload image
- Set status

### Content Types for Announcements

- reminders
- schedules
- advisories
- urgent notices
- department-wide information

---

## 7. Events Module

### Purpose
Manage and present CICT events.

### Features

- Create event
- Edit event
- Delete event
- Upload image
- Set schedule
- Set location
- Set attendee limit
- View event detail
- Join event
- Leave event

### Content Types for Events

- seminars
- workshops
- conferences
- competitions
- organization activities
- orientations

---

## 8. Organization Module

### Purpose
Showcase student organizations.

### Features

- Public organization detail page
- Mission and vision display
- Achievements display
- Team/member display
- Edit basic organization details
- Upload logo
- Upload banner

---

## 9. Member Module

### Purpose
Showcase individual organization members and officers.

### Features

- Public member profile page
- Member bio
- Member photo
- Social links
- Skills display
- Achievement display
- Add member
- Edit member
- Delete member

---

## 10. Media Upload Module

### Purpose
Support image-based content across the website.

### Features

- Upload image
- Validate image type
- Store image reference
- Link image to content records

---

## 11. Public Website Module

### Purpose
Present department content to visitors.

### Features

- Home page
- News page
- Event page
- Organization detail page
- Member detail page
- Responsive navigation
- Theme toggle

---

## 12. Future Enhancement Areas
