# CICT Website Final Product Summary

## What Is the CICT Website?

The CICT Website is a modern department platform for the College of Information and Communication Technology.

It helps the department:

- publish news,
- post announcements,
- manage events,
- showcase organizations,
- present member profiles,
- and manage content through an admin panel.

---

## What Makes It Different?

Many school or department websites are static and difficult to update.

The CICT platform improves this by combining:

```text
Public Website + Admin Management + Structured Content + Role-Based Control
```

This makes it more than a simple static information site.

---

## MVP Focus

The MVP focuses on:

- admin login,
- public website foundation,
- news,
- announcements,
- events,
- organizations,
- members,
- role-based access,
- image upload,
- and core admin management.

---

## Value Model

The platform creates value by:

- improving department communication,
- strengthening digital presence,
- simplifying content management,
- showcasing organizations professionally,
- and making updates more maintainable over time.
