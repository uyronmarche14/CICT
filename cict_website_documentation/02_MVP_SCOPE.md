# CICT Website MVP Scope

## MVP Name
**CICT Website MVP: Public Department Website + Admin Content Management**

## MVP Goal
Launch a usable CICT website where the department can publish and manage content through an admin panel while the public can browse key information.

## MVP Core Promise
Admins can manage important department content without editing code manually, and the public can access updated information through a modern website.

---

# 1. MVP Must-Have Features

## 1. Authentication

### Features

- Admin login
- Admin logout
- Admin session protection
- Profile retrieval

### Purpose
Allow only authorized users to access admin management features.

---

## 2. Admin Dashboard

### Features

- Total users
- Total news
- Total announcements
- Total roles

### Purpose
Give admins a quick overview of system activity and content volume.

---

## 3. User Management

### Features

- Create user
- View users
- Search users
- Paginate users
- Delete user

### Purpose
Allow authorized administrators to manage admin accounts.

---

## 4. Role and Permission Foundation

### Features

- Full Admin role
- Semi Admin role
- Support role
- Backend permission checks
- Role listing

### Purpose
Ensure controlled access to sensitive admin functions.

---

## 5. News Management

### Features

- Create news
- Edit news
- Delete news
- Upload featured image
- Add tags
- Set status
- View public news list
- View public news detail page

### Purpose
Allow the department to publish official articles and updates.

---

## 6. Announcement Management

### Features

- Create announcement
- Edit announcement
- Delete announcement
- Set priority
- Set target audience
- Set expiration
- Upload image

### Purpose
Allow admins to manage short notices and official reminders.

---

## 7. Event Management

### Features

- Create event
- Edit event
- Delete event
- Upload event image
- Set date, time, location, and attendee limit
- View public event list
- View public event detail page
- Join event
- Leave event

### Purpose
Allow the department to post and present events clearly.

---

## 8. Organization Management

### Features

- View organizations
- Edit organization details
- Upload logo
- Upload banner

### Purpose
Allow admins to maintain public organization profiles.

---

## 9. Member Management

### Features

- Add member
- Edit member
- Delete member
- Upload member photo
- View public member profile page

### Purpose
Allow organizations and admins to maintain member and officer visibility.

---

## 10. Public Pages

### MVP Public Pages

- Home
- News
- News detail
- Events
- Event detail
- Organization detail
- Member detail

### Purpose
Provide the minimum complete public website experience.

---

# 3. Not Included in MVP

These are intentionally excluded or only partial in the MVP:

- full About page content
- full Academics page content
- full Admissions page content
- full Student Life page content
- contact submission workflow
- full role creation interface
- full user editing interface
- advanced member timeline and gallery editing
- advanced organization content editing
- audit log dashboard
- system settings page

---

# 4. MVP Success Criteria

- Admin can login successfully.
- Admin can create and manage core content.
- Public users can browse main website content.
- News and events are visible on the public side.
- Organizations and members can be showcased.
- The website is usable on desktop and mobile.
- The system feels like a real working department platform, not just a static prototype.
