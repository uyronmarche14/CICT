# CICT Website Documentation

## Product Name
**CICT Website and Admin Content Management Platform**

## Product Type
Department website, content publishing platform, organization showcase system, event platform, and admin management system.

## One-Line Description
**The CICT Website helps the College of Information and Communication Technology publish official content, showcase organizations and members, manage events, and maintain a modern public-facing digital presence through a protected admin panel.**

## Main Vision
CICT is not just a static department website. It is a growing digital platform where the public website and internal content management system work together.

The platform is designed to become:

- a public information hub,
- a news and announcement platform,
- an event and participation platform,
- an organization and member showcase system,
- an admin-managed department content system,
- and later, a more complete academic and student-information portal.

## Core Product Flow

```text
Admin Login → Create or Update Content → Publish to Public Website → Visitors Browse Information → Students View Events and Organizations → Admin Maintains and Expands Content
```

Start with:

1. Admin authentication
2. Public homepage
3. News management
4. Announcement management
5. Event management
6. Organization pages
7. Member pages
8. Role-based admin protection
9. Media upload
10. Public information pages

**The CICT Website is a modern department platform where official information, student organizations, member profiles, events, and announcements are managed centrally and presented clearly to the public.**
