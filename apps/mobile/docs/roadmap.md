# Roadmap

## Near-Term

- Push notifications for event reminders and important announcements
- Better offline caching and stale-state messaging
- Richer event registration status details
- Student ID or digital pass enhancements

## Mid-Term

- Organization-specific event and update filtering
- Attendance trend summaries for students
- Profile editing if backed by approved APIs

## Longer-Term

- Staff companion mode for attendance operations
- Notification center and in-app inbox
- Mobile-specific analytics and feedback instrumentation
